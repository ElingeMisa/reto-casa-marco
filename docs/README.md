# Casa Marco - Virtual Museum

A responsive web application for an online museum featuring exhibitions, collections, and virtual tours.

## Features

- **Home Page**: Hero section with featured exhibitions and quick information
- **Exhibitions Page**: Detailed view of current and featured exhibitions with virtual tour capabilities
- **Collections Page**: Filterable gallery of museum artifacts and artworks with search functionality
- **Visit Page**: Information about visiting the museum including hours, admission, and booking system
- **About Page**: Museum history, mission, team, and support information

## Interactive Features

- Responsive navigation with mobile hamburger menu
- Collection filtering by category (paintings, sculptures, artifacts, manuscripts)
- Real-time search functionality for collections
- Virtual tour modal system
- Interactive booking form with automatic price calculation
- Smooth scrolling and scroll animations
- Mobile-responsive design

## Technologies Used

- HTML5
- CSS3 (with CSS Grid and Flexbox)
- Vanilla JavaScript
- Gradient backgrounds and modern UI design

## Getting Started

1. Clone this repository
2. Open `index.html` in your web browser
3. Navigate through the different pages using the navigation menu

No build process or dependencies required - just open and run!

## Browser Compatibility
